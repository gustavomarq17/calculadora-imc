# 📊 Calculadora de IMC em C

## 📖 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar lógica de programação utilizando a linguagem C.

A aplicação calcula o Índice de Massa Corporal (IMC) a partir da altura e do peso informados pelo usuário e exibe a classificação correspondente.

## 🚀 Funcionalidades

- Solicita altura e peso do usuário;
- Valida os valores informados;
- Calcula o IMC;
- Exibe a classificação do resultado:
  - Abaixo do peso;
  - Peso adequado;
  - Acima do peso;
  - Obesidade.
